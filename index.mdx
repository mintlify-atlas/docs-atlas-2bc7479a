## Build multiplayer games faster

Colyseus is an authoritative multiplayer game server framework for Node.js. Define your game state on the server — it synchronizes automatically to all clients, delta-compressed and binary-encoded in real time.

## Get started in seconds

```bash
npm create colyseus-app@latest ./my-server
cd my-server
npm start
```

This scaffolds a complete Colyseus server with a sample room, TypeScript configuration, and development tooling ready to go.
